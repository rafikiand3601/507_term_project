Detailed Description

This task is used to control movement of the car.

This task inherits the TaskBase class, and is used to run as a finite state machine. It controls the actions of the motor and servo using shared variables.

void task_car_control::run ( void  )

This method is called to actuate the motor and servo.

This method is called by the RTOS once to run the task loop for ever and ever.

This function works within the FreeRTOS framework. Once it is called, it loops through a switch for as long as the main program is running in a finite state machine. This task uses the shared p_motor_vel and p_servo_pos variables calculate to get the car to move as desired.
