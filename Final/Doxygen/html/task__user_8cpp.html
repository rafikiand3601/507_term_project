Detailed Description

This file contains source code for a user interface task for our ME 507 term project. It is used to set which driving mode is used via a serial interface. It also can be used for debug via serial prints and commands.

const TickType_t ticks_to_delay = ((configTICK_RATE_HZ / 1000) * 5)

This constant sets how many RTOS ticks the task delays if the user's not talking. The duration is calculated to be about 5 ms.
